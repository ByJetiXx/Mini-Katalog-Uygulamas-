# Mini Katalog Uygulaması

Flutter eğitimi kapsamında geliştirilmiş basit bir katalog uygulamasıdır.

## Özellikler

- Ürün listeleme
- GridView kullanımı
- Ürün detay sayfası
- Navigator ile sayfa geçişleri
- Hero animasyonu
- Arama özelliği
- Sepete ekleme simülasyonu
- Material 3 tasarımı

## Kullanılan Teknolojiler

- Flutter SDK
- Dart
- Material Design 3

## Proje Yapısı

lib/
├── data/
├── models/
├── pages/
├── widgets/
└── main.dart

## Çalıştırma

```bash
flutter pub get
flutter run
```
